
Kasturi Holidays Child Theme
A custom child theme for the Kadence theme, Kasturi Holidays, designed to extend your WordPress site with a brand-new custom Elementor widget.

Overview
This repository contains the Kasturi Holidays Child Theme, crafted specifically to add new custom Elementor widgets to your website. By leveraging the power of Elementor and WordPress child themes, this project makes it easy to introduce unique, site-specific functionality without modifying the parent theme’s core files.

Features
Custom Elementor Packages Widget: Seamlessly integrates a new widget into Elementor’s editor, allowing you to add specialized content or features to your pages.

Safe Customization: Built as a child theme, ensuring all custom code is update-safe and separated from the parent theme.

Easy Maintenance: All widget code and customizations are organized for clarity and future updates.

Installation
Download or Clone:

Download this repository or clone it into your WordPress wp-content/themes/ directory.

Activate the Child Theme:

Go to your WordPress dashboard, navigate to Appearance > Themes, and activate Kasturi Holidays Child Theme.

Add the Custom Widget:

The widget is registered automatically when the theme is active. You’ll find it in the Elementor editor panel, ready to use.

How the Custom Elementor Widget Works
The widget is registered from within the child theme’s functions.php file.

Widget code is organized in a dedicated directory (e.g., /widgets/your-widget/), keeping your theme clean and modular.

The widget appears in the Elementor panel with a custom icon and label, making it easy to identify and use.

Example Usage
After activating the theme:

Open any page with Elementor.

Look for the new Kasturi Holidays Widget in the Elementor sidebar.

Drag and drop it onto your page to access its custom controls and display options.
